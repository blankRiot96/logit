## 🛠️ Setup and Install
- Install from `pip`
```
pip install logit-axis
```
OR

- Clone the repository, and install from directory
```
git clone https://github.com/blankRiot96/logit
cd logit
pip install .
```

OR

- Or simply use the `git+` syntax,
```
pip install git+https://github.com/blankRiot96/logit
```
